# Feature Cards

These cards decompose the Kura Metadata Store roadmap into reviewable units.
They describe intent, not implemented behavior.

| Range | Area |
|---|---|
| 001-020 | Core API, MVCC, watches, and leases |
| 021-040 | WAL, snapshots, and durable storage |
| 041-070 | Raft consensus and membership |
| 071-085 | Security, observability, and operations |
| 086-094 | Kura Engine integration |
| 095-099 | Production maturity |

Each implementation still requires a dedicated document under `docs/design/`
before code is written.
