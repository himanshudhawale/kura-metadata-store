# Code of Conduct

Kura Metadata Store welcomes contributors of every experience level.

Be respectful, constructive, and specific. Critique ideas and code rather than
people. Assume good intent, explain unfamiliar concepts, and make room for
questions. Harassment, discrimination, threats, sexualized conduct, deliberate
disruption, and disclosure of another person's private information are not
acceptable.

Project maintainers may edit or remove contributions and restrict participation
when behavior harms the community. Report conduct concerns privately to the
repository owner through their GitHub profile. Reports will be handled as
confidentially as practical.
